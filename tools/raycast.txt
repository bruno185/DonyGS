Given 2 faces f1 and f2. We first verify that the pair has not already been tested. If already tested, we continue.
If no overlap between f1 and f2, continue.
If overlap, we perform ray_cast(f1,f2): if 0: continue; if 1: we mark the pair as tested and continue; if -1: we place f1 just above f2 in the sorted list so that index(f1) = index(f2)+1, and we set a flag to indicate that there was a movement.
Repetition until no more movement.
